class meshmagick.hydrostatics.Hydrostatics(working_mesh, cog=[0.0, 0.0, 0.0], mass=None, rho_water=1023, grav=9.81, verbose=False, animate=False)[source]¶

Bases: object

Class to perform hydrostatic computations on meshes.

Parameters:
  • working_mesh (Mesh) – The mesh we want to work with for hydrostatics computations
  • cog (array_like, optional) – The mesh’s center of gravity coordinates. Default is the origin (0, 0, 0)
  • mass (float, optional) – The mesh’s mass. Default is 0. Unit is Ton.
  • rho_water (float, optional) – The density of water (in kg/m**3). Default is that of salt water (1023 kg//m**3)
  • grav (float, optional) – The acceleration of gravity. Default is 9.81 m/s**2.

Warning

  • At class instantiation, no modification on the mesh position is done and hydrostatic properties are accessible directly for this position. The mesh may not be at equilibrium thought and residual force balance between hydrostatic force and gravity force may be not null. To trig the equilibrium computations, we have to call the equilibrate() method.
  • Note that it is useless to call any method to get the hydrostatic properties up to date as it is done internally and automatically at each modification.
  • Mass unit is the ton !

equilibrate(init_disp=True)[source]¶

Performs 3D equilibrium search.

Parameters:init_disp (bool, optional) – Flag to indicate if the mesh has to be first placed at its displacement. Default is True.
Returns:A code indicating the state of the solver at the end of the computations
Return type:int

Notes

The return code can have the following values:

  • 0 : Failed to find an equilibrium position
  • 1 : A stable equilibrium configuration has been reached
  • 2 : An unstable equilibrium configuration has been reached

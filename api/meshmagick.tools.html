meshmagick.tools module¶

meshmagick.tools.merge_duplicate_rows(arr, atol=1e-08, return_index=False)[source]¶

merge_duplicates(arr, atol=1e-8, return_index=False)

Returns a new node array where close nodes have been merged into one node (following atol).

arr: array_like
numpy array of the coordinates of the mesh’s nodes
atol[optional]: float
the tolerance used to define nodes that are coincident and that have to be merged
return_index: bool
If true, it also returns the array for new indices of vertices
arr
: ndarray
numpy array of the coordinates of the mesh’s nodes where every node is different
newID
: ndarray, optional
numpy arrays of the new new vertices IDs
